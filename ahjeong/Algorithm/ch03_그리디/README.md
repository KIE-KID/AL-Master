# Ch 03 그리디

- 예제 : 거스름돈 
  - 그리디 정당성 : 가지고 있는 동전에서 큰 단위는 항상 작은 단위의 배수 형태이다.
- 실전

  - 큰 수의 법칙
  - 숫자 카드 게임
  - 1이 될 때까지

- 파이썬 문법 복습

  - map()

    : 리스트의 요소를 지정된 함수로 처리해주는 함수

    - map(함수, 리스트) 또는 map(함수, 튜플)
    - Ex) a = list(map(int, a)) : 리스트의 모든 요소를 int 로 반환
